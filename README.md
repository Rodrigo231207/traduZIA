TraduZIA é um tradutor inteligente desenvolvido como projeto pessoal para prática de programação e consumo de APIs.

O projeto permite traduzir textos entre diferentes idiomas de forma simples e intuitiva incluindo captura de áudio por voz, onde o usuário pode falar e o texto é automaticamente convertido para tradução., utilizando JavaScript para integrar uma API de tradução.

  Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Consumo de API
Web Speech API (captura de áudio por voz)

Objetivo do projeto
Este foi meu primeiro projeto pessoal, criado com o objetivo de:
- Praticar lógica de programação
- Aprender a consumir APIs
- Desenvolver interfaces simples e funcionais
- Consolidar conhecimentos em JavaScript
- Trabalhar com reconhecimento de voz no navegador

  Funcionalidades
- Tradução de textos entre múltiplos idiomas
- Interface simples e responsiva
- Interação direta com API externa
Captura de áudio por voz para entrada de texto

  Status do projeto:
Projeto finalizado, com possibilidade de melhorias futuras.
